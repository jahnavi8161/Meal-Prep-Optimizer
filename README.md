🍱 Meal Prep Optimizer

A simple web application that helps users generate personalized meal plans based on dietary preferences and daily calorie goals. The app provides an easy way to plan meals and maintain healthy eating habits.

---

🚀 Live Demo

(After deployment, add your Streamlit link here)

---

✨ Features

- Select diet type (Vegetarian / Vegan / Non-Vegetarian)
- Set daily calorie goal
- Generate a personalized meal plan
- Multi-page Streamlit app
- Clean and interactive user interface

---

🛠️ Tech Stack

▶️ Run Locally

Clone the repository:

git clone https://github.com/jahnavi8161/Meal-Prep-Optimizer.git

Navigate to the project folder:

cd Meal-Prep-Optimizer

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
